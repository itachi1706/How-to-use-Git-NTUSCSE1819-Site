What's required for this

  • A GitHub Account (Create one at GitHub.com)
  • SourceTree Git GUI (Requires an Atlassian Account to register)
  • A Programming IDE
    • Java: Eclipse, IntelliJ Community/Ultimate or jGrasp
    • Python: Anaconda, PyCharm Community/Professional or Python IDLE
    • PHP: PHPStorm or any Notepad application
    • C#/C/C++: Visual Studio or Code::Blocks

Q&A

Why git or GitHub?

Being a Version Control System (VCS) and a VCS provider, this will aloow us to be able to collaborate on the same codebase simultaneously with support for History (through the Commits system) and conflict resolution

We need an account for SourceTree? Can I use something else?

YES! However, this guide is catered to SourceTree but other Git GUI programs will have their own equivalent. Or you can always use git console 😏

I'm actually interested in the Console Command equivalents of the various tools! How do I find them?

Scroll all the way down to the bottom of each page and I will add the console command equivalent beside it if available
