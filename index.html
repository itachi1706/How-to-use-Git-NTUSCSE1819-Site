Q&A

Why git or GitHub?

Being a Version Control System (VCS) and a VCS provider, this will aloow us to be able to collaborate on the same codebase simultaneously with support for History (through the Commits system) and conflict resolution

We need an account for SourceTree? Can I use something else?

YES! However, this guide is catered to SourceTree but other Git GUI programs will have their own equivalent. Or you can always use git console 😏
